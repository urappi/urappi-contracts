### urappi-contracts

Единый репозиторий контрактов для всех микросервисов проекта URAPPI.
Здесь хранятся:

- Протоколы взаимодействия сервисов (gRPC .proto)

- Схемы событий для Kafka (event proto)

- Внешние REST-контракты (OpenAPI)

- Общие типы, правила версионирования и соглашения

Контракты — это источник истины для всех сервисов: gateway, auth, users, posts и любых будущих модулей.
